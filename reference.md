## 数据库

- 路径：`data/duck.db`（可通过 `DUCK_DB_PATH` 修改）
- 表：`duck_profiles`、`draw_history`、`conversation_memory`
- 初始化：首次执行时自动创建

## 核心文件（实现代码）

- `duck.py`：主入口，所有命令实现
- `db.py`：数据库操作（CRUD + 种子记忆）
- `gacha.py`：抽卡系统（32种人格 + 保底贯通）
- `entities.py`：实体识别 + 模糊匹配
- `utils.py`：工具函数（昵称校验 + 相似度）
- `prompts/`：人格标签、称号、自我介绍 JSON
- `config/entities.json`：实体别名配置
- `seed/`：种子记忆（CSV + manifest）
